# Banking System in C

This is a simple console-based banking system written in C.
The program performs basic banking operations like creating an account, depositing money, withdrawing money and checking balance.
All data is stored using file handling so that account details remain saved even after closing the program.

## Features

- Create a new account
- Deposit money
- Withdraw money
- Check balance
- Data stored in file

## Concepts Used

- Structures
- File handling
- Functions
- Switch case

## How to Run

gcc bank.c -o bank
./bank

## Why I Made This

I created this project to practice file handling in C and understand how record management works in real applications.
